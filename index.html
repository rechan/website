Alternate mesh network

Altmesh gives users an anonomouys, open protocol with no central servers or fees.



A platform for innovation.

The first altmesh application allows users to message each other using a safe, annonomouys protocol. The altmesh API exposes everything needed to build rich user experiences in JavaScript.



Device powered networking.

In the future, altmesh will connect device to device in a cloud-like wireless network that is not reliant on servers or data centers. The network data lives on its own.
